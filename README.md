## Web App Installation Instructions

- Clone ToDo-List-app

- Go to project directory

```
git clone https://github.com/abhisheksharma1310/ToDo-List-app.git
cd ToDo-List-app/
```

- Install all packages

- Run webapp

```
npm install
npm run start
```

- For build package

```
npm build
```
